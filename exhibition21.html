Exhibition / 應用展示

Nokia Cloud RAN

    Cloud RAN 的主要市場驅動力：
  1. 更靈活的網路架構 (資源可按需擴展、硬件和軟件的解耦)
  2. 邊緣雲戰略 (新商業模式、端到端切片和低延遲使用案例)
  3. TCO 優勢 (通過自動化及使用通用硬件降低成本)
    諾基亞 Cloud RAN 解決方案提供以下優勢：
  1. 完全雲化 vCU / vDU 更適用于通用硬件平台 (x86 服務器採用最新一代CPU)
  2. CU 和 DU 完全分離，更利于靈活部署
  3. 低延遲應用帶來新收入
  4. 完全雲化架構更適合支援端到端網路切片
  5. 採用效率更高的 CPU 加速技術 (In-Line 加速卡)

    The key market drivers for Cloud RAN are
  1. RAN flexibility and optimization
    a.Cloud provides more flexibility for RAN w.r.t distributed vs centralized topology
    b.Cloud resources can be scaled based on the demand
    c.Decoupling of HW and SW – Generic purpose HW platform can be used for different applications and different vendor SW
  2. Edge cloud strategy (New business models, E2E slicing and low latency use cases)
  3. TCO benefits (via automation, common orchestration, general purpose hardware)
    Nokia Cloud RAN solution offers below benefits
  1. Fully cloudified vCU and vDU with general purpose hardware platform (x86 servers with latest CPU generation)
  2. CU and DU fully disaggregated for flexible deployments (cell site/Far-edge/Edge)
  3. New revenue with low latency applications
  4. More optimized for End-to-end Network Slicing
  5. From look-a-side towards inline L1 acceleration in Cloud RAN
