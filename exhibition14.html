    Exhibition / 應用展示

    Universal Access Platform

    POL Smart Access Network
    POL Smart Access Network provides different physical underlying access network solutions, from fiber optic, copper, cable, FWA and Wi-Fi, and integrates IOT/ICT/IBT smart buildings, communities, healthcare, transportation, etc. A single platform network to meet the high bandwidth needs of customers and the services they need.

    DOCSIS Next Generation Access
    v-CCAP, the upper layer functions run on the virtual host and is managed using software defined network (SDN) and network function virtualization (NFV) concepts. This approach brings a number of significant incremental benefits: it eliminates the noise rise problem in (CMTS)/CCAP physical cable demodulation. It frees up IP investments by preserving the existing video requirements Edge QAM (EQAM) infrastructure. The node is provided with 100% digital signal quality on day 1. There is no need to simulate optical components. Convert legacy video to Ethernet using a unique video engine to ensure 100% investment protection for existing deployments of EQAM. v-CCAP also accelerates and simplifies service deployment - Zero-touch configuration simplifies network configuration to provide faster return on investment (ROI). v-CCAP helps global operators solve their existing hybrid fiber-optic coaxial cable (HFC) infrastructure.

    被動光纖區域智慧接取網路
    諾基亞智慧接入平台提供不同物理底層接取網路解決方案，從光纖、銅纜、電纜、FWA 和 Wi-Fi，並整合 IOT/ICT/IBT 智慧大樓、社區、醫療、交通等實現多元化網路單一平台網路，滿足客戶高頻寬需求及所需服務。

    DOCSIS 下一代接取網路
    v-CCAP，上層功能在虛擬主機上運行，並使用軟體定義網路 (SDN) 和網路功能虛擬化 (NFV) 概念進行管理。這種方法帶來了許多顯著增量收益：它消除了在 (CMTS)/CCAP 物理電纜解調產生噪升問題；它通過保留現有視頻需求 Edge QAM (EQAM) 基礎結構，為 IP 投資騰出資金。初始為節點提供 100% 數位信號品質，不需模擬光學元件；利用特有視頻引擎將傳統視頻轉換為乙太網，確保對現有部署 EQAM 提供 100% 投資保護。v-CCAP 還加速並簡化了服務部署 - 零接觸配置簡化網路配置，以提供更快投資回報 (ROI)。v-CCAP 幫助全球營運商解決其現有混合光纖同軸電纜 (HFC) 基礎設施。
