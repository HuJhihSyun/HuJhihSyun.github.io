Exhibition / 應用展示

Nokia Adaptive Cloud Networking

Adaptive Cloud Networking 為服務供應商提供了改造網路所需的工具和功能，以應對 5G 時代快速變化的需求，增強數據中心的結構以及對邊際雲的管理能力，並使整體網路轉變為更敏捷和自動化。

Adaptive Cloud Networking 利用 NOS 系統 (Network Operating System) 的開放性與模組化架構的特性，有效地讓使用者透過多元的管理系統 (如監控系統、分析系統、自動化...等) 來分析網路數據，讓使用者能更好地掌握整個網路。

Adaptive Cloud Networking 的另一個優勢為基於 NetOps 的自動化能力。通過利用 NetOps 原則，可以自動地執行數據中心的生命週期的每個階段 (從 Day 0 的設計階段；Day 1 的部署階段到 Day 2+ 之後的維運) ，從而改變原有的維運結構，減少人為錯誤，並提供更一致的服務。

Adaptive Cloud Networking gives service providers the tools and capabilities they need to transform their network to respond to the unpredictable demands of the 5G era. This cloud-native solution supercharges the data center fabric and seamlessly extends operations to the edge clouds. It allows service providers to transform their network into a consumable, agile and automated whole that embraces the cloud-native era and turns 5G into a revenue and growth opportunity.

Adaptive Cloud Networking makes it easy for the operator and northbound clients such as monitoring systems, analytics platforms, automation platforms, and device configuration platforms to consume network data. One of the foundational drivers behind consumable networking is the solution’s network operating system (NOS), which provides ubiquitous visibility and control with its open, extensible, and model-driven architecture.

Consumable networks give service providers greater visibility into their network data across the entire network, enable them to access it more easily, and empower their operations with a rich data set.

One of the strengths of Adaptive Cloud Networking is its complete set of NetOps-based automation capabilities. By Leveraging NetOps principles, the solution can revolutionize operations by automating each phase of the data center fabric’s operational lifecycle from Day 0 design to Day 1 deployment to Day 2+ operations.

A data center network that adopts these advanced automation capabilities will allow service providers to improve the productivity of network operations, simplify operational tasks, reduce human errors, and deliver a more consistent service.
