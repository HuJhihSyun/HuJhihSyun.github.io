REGISTRATION

Hardik K Gohil

Global Head of Mobile Packet Core Business Development, Nokia

Hardik, an alumni of the University of Southern California joined Alcatel's IP division from College as a software engineer for IP portfolio. He was one of the first engineers to join the Packet Core development team and since then has been part of Pre-Sales, Product management at various levels. Currently he heads the Mobile Packet Core Business Development, ION HQ.

畢業於南加州大學的 Hardik，大學時便加入了阿爾卡特公司的 IP 部門，擔任 IP 方案組合的軟體工程師。他是首批加入 Packet Core 開發團隊的工程師之一，此後擔任售前顧問、產品管理部的各項職務。他目前負責總部 IP 與光纖網路 (ION) 部門的 Mobile Packet Core 業務開發。

Topic Abstract

議程介紹

Access Agnostic Mobile Packet Core for 5G and evolution to fixed mobile convergence

Universal adaptive Mobile Packet core — focus on access agnostic packet core and it’s evolution to fixed mobile network convergence/micro-serivce benefit on Packet Core.

通用可適應性行動封包核心 (Universal Adaptive Mobile Packet Core) — 主要討論支援任一接取方式 (access-agnostic) 的封包核心，及其演進到封包核心上固定行動網路融合/微服務的優勢。
