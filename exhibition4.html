Exhibition / 應用展示

Nokia Core SaaS

諾基亞 5G SA 核心網元可在任何雲平台上實現擴展性和敏捷性。這種”軟件即服務”的商業模式在 IT業界 中已經很常見，我們在電信領域也有很多這樣的例子。 我們正在將這個概念，引入在以軟體主導的 5G 核心網路。

這個概念演示是關於一個專案交付框架，該框架將成為從訂購 / 到交付運營以 SaaS 的基礎的核心網路。 交付核心網路的整個概念改變了服務上線的時間，作為 SaaS 服務供應商的諾基亞提供了運營的主要責任及肩負雙方預先協議的 SLA。

在本演示中，我們將展示一個使用諾基亞核心 SaaS 交付框架的 5G SA 核心網路在第三方公有雲實現實例。

Nokia 5G Standalone Core on any cloud for scaling and agility.

This business model we call Software as a Service is familiar in IT, and we have also many instances for that in the telco world. We are working on this concept in one of the most software dominant element this is the 5G core.

This conceptual demo is about is projecting a delivery framework which will be a base of ordering / operating Core SaaS. The whole concept of delivering core network changes to absolute fast Time-To-Market, and operation’s main responsibility and providing pre-defined SLA shoulders on SaaS provider which is Nokia.

Nokia uses 3rd party public cloud to implement 5G core network functions. In this demo we will present an absolute greenfield instantiation of a 5G SA core using Nokia Core SaaS delivery Framework.
